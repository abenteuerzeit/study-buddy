# CCNA Prep

**CCNA Prep** is a web-based interactive platform designed to help users prepare for the Cisco Certified Network Associate (CCNA) exam. It combines adaptive flashcards, spaced repetition, and learning statistics into a single dashboard for effective self-paced learning.

## Features

- **Adaptive Flashcards**: Learn using dynamic flashcards tailored to your knowledge level.
- **Spaced Repetition System (SRS)**: Built-in SM-2 algorithm to optimize your study schedule and memory retention.
- **Learning Filters**: Filter questions by category, source, or chapter to focus your efforts.
- **Fuzzy Search**: Search through flashcards by keyword.
- **Image-Based Questions**: Supports visual learning through symbol recognition.
- **Progress Statistics**: Visualize your mastery, streaks, review due cards, and overall accuracy.
- **Import/Export Progress**: Backup or transfer your learning progress via JSON files.
- **Dark UI Theme**: Sleek and modern interface built with Tailwind CSS.

## How to Use

1. **Open `index.html` in your browser** – no installation needed.
2. **Study Using Flashcards** – Choose filters, answer questions, and improve your score.
3. **Export Your Progress** – Use the Export button to save your study progress as a `.json` file.
4. **Import Existing Progress** – Restore your previous session using the Import feature.

## Local Storage

The app uses your browser’s `localStorage` to track spaced repetition data and stats. No data is sent to any server.
