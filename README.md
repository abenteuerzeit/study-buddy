# Study Buddy

**Study Buddy** is a web-based interactive platform designed to help users memorize information for any knowledge domain. It combines adaptive flashcards, spaced repetition, and learning statistics into a single dashboard for effective self-paced learning.

## Features

- **Adaptive Flashcards**: Learn using dynamic flashcards tailored to your knowledge level.
- **Spaced Repetition System (SRS)**: Built-in SM-2 algorithm to optimize your study schedule and memory retention.
- **Learning Filters**: Filter questions by category, source, or chapter to focus your efforts.
- **Fuzzy Search**: Search through flashcards by keyword.
- **Image Support**: Supports visual learning through associated images.
- **Progress Statistics**: Visualize your mastery, streaks, review due cards, and overall accuracy.
- **Import/Export Progress**: Backup or transfer your learning progress via JSON files.
- **Import/Export Flashcards**: Easily manage your flashcard data.
- **Flashcard Management (CRUD)**: Add, Edit, and Delete flashcards directly within the app.
- **Dark UI Theme**: Sleek and modern interface built with Tailwind CSS.

## How to Use

1.  **Open `index.html` in your browser** – no installation needed.
2.  **Manage Flashcards** - Go to Settings to add, edit, or delete your flashcards.
3.  **Study Using Flashcards** – Choose filters, answer questions, and improve your score.
4.  **Export Your Progress** – Use the Export button to save your study progress as a `.json` file.
5.  **Import Existing Progress** – Restore your previous session using the Import feature.
6.  **Import/Export Flashcards** - Manage your flashcard data files.

## Local Storage

The app uses your browser’s `localStorage` to track spaced repetition data and stats. No data is sent to any server.
